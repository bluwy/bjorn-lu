# My website

Built with SvelteKit and WindiCSS.

## Development

```bash
# Install dependencies
$ pnpm install

# Start dev server
$ pnpm dev

# Build production site
$ pnpm build

# Preview production site
$ pnpm preview

# Format code with Prettier
$ pnpm format
```
